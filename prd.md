# 📝 PRD: 내 싸인 만들기 ! (My Signature Maker)

## 1. 제품 정의 (Product Definition)
사용자 이름(3글자 이내)과 선택한 분위기 카테고리를 조합하여, 실제 펜으로 쓴 듯한 고품질 디지털 싸인 이미지를 생성하고 사용자의 기기(모바일 갤러리)에 저장해주는 **단발성 클라이언트 사이드 미니앱**입니다.

---

## 2. 핵심 목표 및 제약 조건
- **모바일 전용 (App-in-Toss):** 토스 앱 내 웹뷰 환경에서 최적의 퍼포먼스를 내도록 설계.
- **Zero-Backend:** 별도의 서버나 데이터베이스 없이 브라우저 단에서 모든 로직(생성, 변환)을 처리.
- **No-Auth:** 로그인 및 사용자 인증 절차를 완전히 배제하여 접근성 극대화.
- **단순한 Flow:** 프리뷰 과정 없이 [입력 -> 생성 -> 저장]의 직관적인 단계로 구성.

---

## 3. 사용자 여정 (User Journey)
1. **입력:** 토스 미니앱 진입 후 이름(최대 3글자) 입력.
2. **선택:** 5가지 분위기(클래식, 시그니처, 미니멀, 핸드라이팅, 볼드) 중 1개 선택.
3. **생성:** '싸인 만들기' 버튼 클릭 시 결과 화면으로 전환.
4. **확인 및 저장:** 애니메이션과 함께 등장하는 최종 싸인을 확인하고 '이미지 저장하기' 버튼 클릭.

---

## 4. 상세 기능 (Functional Requirements)

### 4.1 이름 및 카테고리 설정
- **이름 입력:** TDS `TextField`를 사용하며 최대 3글자 제한 (`maxLength={3}`).
- **카테고리 선택:** TDS `Chips` 또는 `SegmentedControl`을 활용해 5종 중 택 1.
- **상태 관리:** `Zustand`를 이용해 입력된 이름과 선택된 스타일 ID를 임시 보관.

### 4.2 싸인 생성 및 이미지화
- **렌더링:** 선택한 카테고리에 맞는 폰트 및 CSS Transform(회전, 기울기)을 적용한 DOM 생성.
- **이미지 변환:** `html-to-image` 라이브러리를 사용하여 브라우저 내에서 직접 PNG 파일 생성.
- **애니메이션:** `framer-motion`의 `path` 또는 `opacity` 애니메이션을 통해 획이 그려지는 효과 구현.

### 4.3 저장 기능
- **Web API 활용:** 생성된 이미지 Blob 데이터를 브라우저 다운로드 기능을 통해 모바일 기기 갤러리에 저장.
- **토스 SDK 연동:** 저장 성공 시 TDS 스타일의 토스트 메시지 노출.

---

## 5. 기술 스택 (Technical Specs)
- **Runtime:** Node.js v20.19.6
- **Build Tool:** Vite v7.3.1 (React + TypeScript)
- **UI Framework:** TDS (Toss Design System) 디자인 가이드 준수
- **Core Libraries:**
  - `framer-motion`: 싸인 드로잉 및 화면 전환 애니메이션.
  - `html-to-image`: DOM 요소를 이미지 파일로 변환.
  - `zustand`: 가벼운 클라이언트 전역 상태 관리.
  - `lucide-react`: UI 보조 아이콘.

---

## 6. UI/UX 가이드라인 (Mobile Only)
- **입력 화면:** 깔끔한 화이트 배경에 입력 폼과 카테고리 버튼 배치.
- **결과 화면:** 싸인이 돋보이도록 중앙에 집중된 레이아웃. 종이 질감(Texture)의 배경 요소 추가.
- **햅틱 피드백:** 버튼 클릭 시 물리적인 진동 효과를 주어 '도장을 찍는 듯한' 경험 제공.

---

## 7. 로드맵 (Roadmap)
- **Phase 1 (현재):** 기본 생성 및 저장 기능 구현 (MVP).
- **Phase 2 (수익화):** `@apps-in-toss/web-framework` 연동을 통한 저장 전 전면 광고 노출 로직 추가.
- **Phase 3 (고도화):** 잉크 색상 변경 및 필기체 폰트 라이브러리 확장.

---

## 8. 데이터 보안
